CSS-3 RADIAL-GRADENTS...!

In a radial gradient color emerge from a single point and smothely spread outward in acircular or elliptical shape rather than fading from one colour to another in single direction as with liner gradients. The basic syntax of creating a radial gradient can be given with

syntax : radial-gradient(shape size at position, color-stop1, color-stop2...)
